## 项目基础信息
- 开发框架：基于 Chappie-II 应用逻辑，移植至 **ESP-IDF V5.5.3**
- 硬件主控：**ESP32-S3（16MB Flash + 8MB PSRAM）**
- 推荐模组：ESP32-S3-WROOM-1U-N16R8（外置天线版本）
- 编译方式：VS Code 直接导入 `src` 文件夹，即可编译、烧录
- 编译时会自动下载`managed_components`文件夹

---

## 现有功能
- 网络：WIFI 连接、网络天气获取、网络时间同步
- **手机传图**：徽章开 WiFi 热点，手机浏览器打开网页即可上传 GIF / JPG 到 SD 卡（免装 App，iOS / 安卓通用）
- 显示：屏幕自动息屏、SD 卡内 GIF / JPG 图片**自动扫描轮播**（数量不再限制，左右滑动切换）
- 小游戏：内置电子木鱼、2048

---

## 手机传图使用说明
1. 在桌面进入「Transfer / 手机传图」App
2. 屏幕显示热点信息，手机连接 WiFi 热点：
   - 名称：`CyberBadge_Pic`
   - 密码：`12345678`
3. 手机浏览器打开 `http://192.168.4.1`
4. 在网页里选择 jpg / gif（可多选）点击上传，可查看 / 删除 SD 卡现有图片
5. 按 HOME 键退出，进入「ShowGIF」即可左右滑动查看刚上传的图片

> 实现：徽章侧用 ESP-IDF 自带的 `esp_http_server` 提供上传网页并把文件写入 `/sdcard`；
> 进入该 App 会临时切到 AP 热点模式，退出后自动释放。相关代码见
> `CyberBadgeUI/Apps/App_Transfer/`。

## 已知问题 & 优化建议
- **RTC 时钟**：使用 ESP32-S3 内置 RTC，睡眠关机后时间误差较大，建议每次开机后在设置里同步网络时间
- **电量检测**：使用内置 ADC 实现，误差较大，仅作粗略显示；如需调试电量信息，可在 `CyberBadgePower.hpp` 中取消日志注释
